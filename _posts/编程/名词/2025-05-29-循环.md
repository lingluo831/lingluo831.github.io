循环
### 1. for 循环
- **一句总结**：适用于已知循环次数的场景，结构紧凑，包含初始化、条件判断、递增操作。
- **示例**：
```cpp
for (int i = 0; i < 5; ++i) {
    std::cout << i << std::endl;
}
```
> 输出 0 到 4，每次循环 i 自增 1。

---

### 2. while 循环
- **一句总结**：适用于循环次数未知，仅在满足条件时继续执行的场景。
- **示例**：
```cpp
int i = 0;
while (i < 5) {
    std::cout << i << std::endl;
    ++i;
}
```
> 只要 i 小于 5 就一直循环。

---

### 3. do...while 循环
- **一句总结**：至少执行一次循环体，然后在条件满足时继续执行，适合“先做再判断”的场景。
- **示例**：
```cpp
int i = 0;
do {
    std::cout << i << std::endl;
    ++i;
} while (i < 5);
```
> 即使初始条件不满足，循环体也会先执行一次。

---

### 拓展：范围-based for 循环（C++11及以后）
- **一句总结**：用于遍历数组、容器等集合，更简洁安全。
- **示例**：
```cpp
std::vector<int> nums = {1, 2, 3};
for (int n : nums) {
    std::cout << n << std::endl;
}
```
> 依次输出容器中的每个元素。
